## 🛠️ Tech Stack & Performance

This project is built as a high-performance static Progressive Web App (PWA), optimized specifically for **GitHub Pages** and mobile-first delivery.

- **Frontend**: Vanilla HTML5, Modern CSS3 (Glassmorphism & GPU Accelerated Animations).
- **Core Logic**: Modern JavaScript (ES6+), Intersection Observer API for Lazy Loading.
- **Offline Capabilities**: Service Workers for asset caching and PWA support.
- **Optimization Strategy**:
  - **Dynamic Lazy Loading**: Videos and heavy assets are only loaded when nearing the viewport, saving up to 80% of initial bandwidth.
  - **GPU Acceleration**: Using `will-change` and hardware-accelerated transforms to ensure 60FPS on mobile.
  - **Lite-Mode Detection**: Automatic hardware detection to disable heavy effects (like `backdrop-filter`) on low-end devices.

## 📦 Deployment & Setup

The site is configured for zero-config deployment on **GitHub Pages**:

1. **Clone the repo**: `git clone https://github.com/5410m0n0c001/INIT-IDEA.git`
2. **Local Preview**: Open `index.html` with any live server (e.g., Live Server VS Code extension).
3. **Deploy**: Push changes to the `main` branch. The `.nojekyll` file ensures GitHub doesn't ignore underscored folders or bypass custom logic.
